# Vue 3 + Vite

This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.

Learn more about IDE Support for Vue in the [Vue Docs Scaling up Guide](https://vuejs.org/guide/scaling-up/tooling.html#ide-support).


## Notes

Замечания по шрифтам: в Фигме шрифт DM Sans, у которого нет официального кириллического начертания, поэтому я использовала Open Sans, как максимально похожий.

В дизайне есть крестик в правом верхнем углу, но в задании не указано, какая у него должна быть функциональность. Поэтому я оставила его без логики, чтобы не добавлять поведение, не описанное в требованиях.

Еще я не совсем поняла, как быть с кнопками в мобильной версии, в дизайне на первом шаге Отменить и Далее, а на втором - Назад и Далее. Я решила оставить пользователю возможность сбросить форму на любом шаге. 

Форму для дополнительной информации разрешила оставить пустой, т.к. обычно ее заполнение опционально.
